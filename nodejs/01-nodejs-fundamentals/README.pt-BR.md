## Iniciando

```bash
# Executar aplicação
npm run dev
```

## Comandos executados

```bash
# Inciando projeto
 npm init -y

# Executar aplicação
node src/server.js

# Executar aplicação
 node --watch src/server.js
```

<br>

## Conceitos

- **Non Blocking IO:** _Input_ e _Output_ não bloqueante
